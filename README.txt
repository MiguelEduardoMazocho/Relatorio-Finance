📈 Relatório Financeiro

Bem-vindo ao projeto Relatório Financeiro!Este projeto permite inserir dados financeiros para serem exibidos em um gráfico de pizza dentro de um executável.


✨ Funcionalidades

📊 Organização de Dados Financeiros

🗂️ Classificação e separação de receitas e despesas de forma organizada.

🧮 Cálculo de Valores

➕ Soma total de receitas.

➖ Soma total de despesas.

🧾 Cálculo do saldo final (receitas - despesas).

📝 Formatação de Relatório

📋 Preparação do conteúdo em formato textual organizado, com listas, tabelas ou descrições claras das transações.

📊 Cria um gráfico de pizza ilustrando a distribuição dos dados.


🚀 Como Executar o Projeto

Clone o repositório:

git clone https://github.com/MiguelEduardoMazocho/Relatorio-Finance.git

Acesse a pasta do projeto:

cd Relatorio-Finance

Execute o programa acessando a pasta "dist" e abrindo o arquivo "relatory.exe"

📊 Resultado

O script irá gerar:

Um gráfico de pizza com os dados inseridos pelo usuário.
